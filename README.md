## ✨ Features

- 🎬 **YouTube Showcase** — 8 embedded music videos with click-to-expand inline player
- 👤 **Team Section** — Real studio photos with split-panel bio cards
- 🎹 **Sticky Music Player** — Mini player with real track names cycling at the bottom
- 🌊 **Animated Waveform** — Live SVG waveform bars in the hero section
- 🧭 **Navigation** — Active section tracking via IntersectionObserver + glassmorphism
- ✉️ **Contact Form** — Reach out for brand projects
- 📱 **Fully Responsive** — Mobile-first design
